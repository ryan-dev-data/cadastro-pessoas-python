# Cadastro de Pessoas em Python

Projeto desenvolvido para praticar lógica de programação com Python.

## Funcionalidades
- Cadastro de múltiplas pessoas
- Verificação de maioridade
- Contagem de maiores e menores de idade
- Exibição de relatório final

## Tecnologias
- Python 3

## Objetivo
Projeto de estudo com foco em lógica, listas, estruturas de repetição e condicionais.
